# Beepo - Pomodoro Timer

## Overview

This is my very own Pomodoro timer, designed to help me stay focused and productive by following the principles of the Pomodoro technique.

Later in development, I realized that the look of the buttons somewhat resembled those retro alarm clocks with extruded buttons, so I decided to stick with that vibe.

Go ahead and give it a try—it might help boost your productivity!

---

## Technologies Used

- **HTML**
- **CSS**
- **JavaScript**
- **GitHub Pages**

---

## Features

- **Adjusting Timer:** Can increment and decrement the value by hovering and scrolling on the hour, minute, or second part of timer.
- **Themes:** Switch to light and dark mode by clicking on the smiling face (hooman).
- **Streaks**: Count the number of pomodoros completed. Goes back to zero every session.
- **Tomatoes**: Tomatoes are representation of _streaks_, they bounce across the page.
